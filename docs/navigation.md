# Карта навигации и маршруты

## 1. Текстовая карта навигации

```text
Главная
├── Каталог услуг
│   └── Детальная страница услуги
├── Портфолио
├── Авторизация / Регистрация
├── Личный кабинет
│   ├── Профиль
│   └── Мои заявки
│       └── Детальная страница заявки
└── Админ-панель
    ├── Дашборд
    ├── Услуги (CRUD)
    ├── Портфолио (CRUD)
    └── Заявки (просмотр + смена статуса)
```

## 2. Маршрут гостя

1. Главная.
2. Переход в каталог услуг.
3. Фильтрация/поиск.
4. Открытие детальной страницы услуги.
5. Переход к регистрации или авторизации для отправки заявки.

## 3. Маршрут авторизованного пользователя

1. Авторизация.
2. Главная или каталог.
3. Выбор услуги и отправка заявки.
4. Переход в личный кабинет.
5. Просмотр списка заявок и текущих статусов.

## 4. Маршрут администратора

1. Авторизация под ролью `admin`.
2. Переход в админ-панель (дашборд).
3. Управление услугами и портфолио (CRUD).
4. Переход в раздел заявок.
5. Изменение статусов заявок и контроль обработки.

## 5. Связь страниц с требованиями

| Страница | Требование |
| --- | --- |
| Главная | Публичная презентация студии, CTA на заявку |
| Каталог услуг | Просмотр услуг, поиск, фильтрация (AJAX) |
| Детальная услуга | Полное описание услуги и переход к заявке |
| Портфолио | Демонстрация кейсов, стеков и связанных услуг |
| Авторизация / Регистрация | Доступ в защищенные разделы |
| Личный кабинет | Просмотр своих заявок и статусов |
| Админ-панель | CRUD услуг/портфолио, управление заявками |

## 6. Правила доступа

1. Публичные страницы доступны всем.
2. Личный кабинет доступен только авторизованным пользователям.
3. Админ-панель доступна только роли `admin`.
4. При отсутствии прав система перенаправляет на страницу входа.
